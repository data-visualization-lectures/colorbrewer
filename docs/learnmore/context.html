ヒント：郡の境界線をオフにするか、白色にしてみてください。大きな違いに気づきますか？地図を囲む背景を変更して、周囲の環境によって色がどのように変化するかを確認してください。

配色の外観と堅牢性は、地図上に配置される他の要素や、色を表示しようとしている背景の影響を部分的に受けます。線の色のわずかな違いや、他の地図要素（ラベルなど）の存在は、配色の外観に大きな影響を与えるため、最終的な配色を決定する前に、ここでこれらのオプションを試してください。

都市や道路をオーバーレイして、選択したエリアカラーでテキストやシンボルがどの程度読みやすいかを最初に確認してください。ここで選択した例は高速道路と都市ですが、他の線や文字が地図上でどのように機能するかについて良いアイデアを得られるはずです。

また、グレースケールのDEMも提供しているため、色を他の基礎地図データと組み合わせたときに何が起こるかを確認できます。一般的に、色は区別しにくくなり、より少ないデータクラスを使用する必要があります。
