Color Theory (wikipedia)

データの階級数を選択することは、地図デザインにおいて重要な要素です。階級数を増やすと、データの一般化が減少し、より「情報量の多い」地図になります。しかし、階級数が多すぎると、地図の読者が情報に圧倒され、分布の一般的な傾向を見ることができなくなる可能性があります。さらに、階級数が多いと地図の判読性が損なわれる可能性があります。階級数が多いほど、区別が難しくなる色が必要になります。

多くの地図製作者は、コロプレス図には5〜7つの階級を使用することを推奨しています。等値線図や、非常に規則的な空間パターンを持つコロプレス図では、類似した色が隣り合って表示されるため、区別しやすくなり、より多くの階級数を安全に使用できます。
